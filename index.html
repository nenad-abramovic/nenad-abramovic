Currently supported commands:


open trgovac - opens Trgovac project.

Supported parameters: trgovac, forum, linkedin, github and cv.


weather Belgrade - gets current temperature in Belgrade.
Supported parameters: more known cities in the world.


current time - gets current time.
Supported parameters: time and date.


timer 10 - sets timer for 10 seconds.
Supported parameters: number of seconds.


2 + 3 - 5 - calculates expression.
